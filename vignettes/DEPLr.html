Introduction

DEPLr in an implementation of a Monte Carlo simulation depletion model applied to octopus. This package allow the user to project the population of a stock, or different stocks of octopus with no interaction, in the future and get a summary of the state of the population. It also provides different outputs like:

  1. The path of the abundance of the stock along the fishing season
  2. The path of the biomass for each zone
  3. The catch in number and weight along the weeks of the fishing season for each zone and fleet
  4. And the cpue of each fleet in each zone

Input file

There is an example of the input file in the package that could be uploaded using the DEPLSim method. This input file include all basic elements needed to run the simulation model. A full description of the input file could be found in Appendix A:

As you can see, there are eight main feilds in the DEPLSim object which we are going to present briefly:

  1. controle: This slot is a DEPLPar object which contains some parameters of the simulation model. To look further on it type controle(sim). We used knitr::kable just for formating the table to look like this:
knitr::kable(controle(sim))
Parameter Description Value Unit
cstNM Constant natural mortality(1=TRUE,0=FALSE) 0.00e+00
valNM Value of natural mortality per week 1.37e-02
proErr Process error 1.00e-01
obsErr Observation error 1.00e-01
nbrSim Number of simulations 1.00e+02
season Length of fishing season in weeks 1.90e+01
nbrFleet Number of fleets 3.00e+00
nbrZone Number of Zones 3.00e+00
discard Difference between survey and begining of fishing season 5.00e+00
recTot Total recruitment 5.00e+06

  1. params: this slot is a DEPLPar object which contains the parameters of the Length-Weight relationship and the parameters of the Growth function of the octopus.

  2. zones: this slot is a DEPLZone object. To have access to it we can use the following script:

zone <- zones(sim)
# Get the names of zones
names(zone)
## [1] "A-Côte"  "A-Large" "B"
# plot the proportion of initial abundance in each zone
plot(zone)

To get the fleets exploiting the zones we type:

# Get the fleets exploiting the zones 
fleet <- fleet(zone)
# To obtain names of each fleet
names(fleet)
## [1] "Artisanal" "Côtier"    "Hauturier"
# To obtain the catchability of each fleet
catchability(fleet)
## [1] 3.0e-05 3.4e-05 9.0e-05
# To display the Fleets with their catchabilities in a data.frame
as.data.frame(fleet)

## 1 Artisanal      3.0e-05
## 2    Côtier      3.4e-05

The last five fields of the DEPLSim object sim are _catch.n__, catch.wt, stock.n, stock.wt and cpue. These could be accessed with:

catch.n(sim)
## [1] NA
catch.wt(sim)
## [1] NA
stock.n(sim)
## [1] NA
stock.wt(sim)
## [1] NA
index(sim)
## [1] NA

As you can see, these objects are empty, so to have them full, we have to run the simulation model using run method:

# sim <- run(sim)

After runing the model you can se that the empty feilds are now having values. To ge summaries of the stock abundance, biomass, the catch in numbers and weight, and the cpue we use the following script:

# Computing a summary of the catch in numbers
catchN <- summary(object = sim,"catch.n")
catchWT <- summary(object = sim,"catch.wt")
stockN <- summary(object = sim,"stock.n")
stockWT <- summary(object = sim,"stock.wt")
cpue <- summary(object = sim,"cpue")
